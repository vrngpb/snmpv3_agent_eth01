# SNMP-агент на базе WT32-ETH01

Мониторинг серверного помещения по **SNMPv3 authPriv**: температура, протечка воды, открытие дверей, удалённая смена сетевых настроек.

## Возможности

- **Температура** — DS18B20 по 1-Wire, опрос каждые 2 секунды
- **Протечка воды** — Gidrolock WSP через транзисторную развязку
- **Датчики дверей** — 2 геркона (сухой контакт)
- **Светодиодная индикация** — сеть, авария
- **SNMPv3 authPriv** — SHA-1 аутентификация + AES-128 шифрование
- **Стандартный MIB2** + кастомный enterprise MIB (`.1.3.6.1.4.1.9999.1`)
- **Удалённая смена IP/шлюза/маски** — SNMP SET с валидацией и авто-перезагрузкой
- **Сброс к заводским** — удержание кнопки 5 секунд (стирает NVS)
- **Аппаратный watchdog** — автоматическая перезагрузка при зависании задач

## Логика работы датчиков

### DS18B20 (температура)

Питание 3.3V, данные по 1-Wire на GPIO 32. Между VDD и DQ обязателен резистор 4.7 кОм.
Значение публикуется в SNMP как `INTEGER` (температура × 10, например 246 = 24.6 °C).

### Gidrolock WSP (протечка)

Датчик питается от 5V. Без воды — база транзистора (2N3904) притянута к GND, транзистор закрыт, GPIO 33 — логическая 1 (PULLUP). При протечке вода замыкает контакты, транзистор открывается, GPIO замыкается на GND — логический 0.

> [!CAUTION]
> Никогда не подключайте Gidrolock WSP напрямую к GPIO! Напряжение шлейфа 5–24V сожжёт порт. Только через транзистор.

### Герконы (двери)

Дверь закрыта: геркон замкнут на GND → GPIO = 0. Дверь открыта: геркон разомкнут → GPIO = 1 (PULLUP). SNMP возвращает 1 при открытии (тревога).

## SNMP-команды

### Linux (net-snmp)

```bash
# Прочитать все показатели
snmpwalk -v3 -l authPriv -u username \
  -a SHA -A passphrase \
  -x AES -X passphrase \
  192.168.2.75 .1.3.6.1.4.1.9999.1

# Температура
snmpget -v3 -l authPriv -u username \
  -a SHA -A passphrase \
  -x AES -X passphrase \
  192.168.2.75 .1.3.6.1.4.1.9999.1.1.0

# Изменить IP (устройство перезагрузится через 1 сек)
snmpset -v3 -l authPriv -u username \
  -a SHA -A passphrase \
  -x AES -X passphrase \
  192.168.2.75 \
  .1.3.6.1.4.1.9999.1.10.0 s "192.168.2.80" \
  .1.3.6.1.4.1.9999.1.11.0 s "192.168.2.65" \
  .1.3.6.1.4.1.9999.1.12.0 s "255.255.255.224"
```

## Мониторинг

### Linux — bash-скрипт (`monitor.sh`)

Выводит строку каждые 2 секунды. Требует пакет `snmp` (`apt install snmp`).

```bash
chmod +x monitor.sh
./monitor.sh
```

```bash
#!/bin/bash

HOST="192.168.2.75"
COMMUNITY="-v3 -l authPriv -u username -a SHA -A passphrase -x AES -X passphrase"

get() { snmpget -Oqv $COMMUNITY "$HOST" "$1" 2>/dev/null; }

while true; do
    TEMP=$(get 1.3.6.1.4.1.9999.1.1.0)
    HEAP_PCT=$(get 1.3.6.1.4.1.9999.1.4.0)
    LEAK=$(get 1.3.6.1.4.1.9999.1.6.0)
    DOOR1=$(get 1.3.6.1.4.1.9999.1.7.0)
    DOOR2=$(get 1.3.6.1.4.1.9999.1.8.0)

    TEMP_C=$(awk "BEGIN {printf \"%.1f\", $TEMP/10}")
    [ "$LEAK"  = "1" ] && LEAK_S="ТРЕВОГА" || LEAK_S="ОК"
    [ "$DOOR1" = "1" ] && D1_S="ОТКРЫТА"  || D1_S="закрыта"
    [ "$DOOR2" = "1" ] && D2_S="ОТКРЫТА"  || D2_S="закрыта"

    echo "[$(date '+%H:%M:%S')]  Температура: ${TEMP_C}°C  Heap: ${HEAP_PCT}%  Протечка: ${LEAK_S}  Дверь1: ${D1_S}  Дверь2: ${D2_S}"

    sleep 2
done
```

### Windows / Python — скрипт (`test_snmp.py`)

**Установка зависимостей:**

```
python -m pip install "pyasn1==0.4.8" "pysnmp==4.4.12"
```

> Более новые версии pysnmp (6.x) имеют несовместимый API. Указанные версии обязательны.

**Запуск:**

```
python test_snmp.py
```

```python
import time
from pysnmp.hlapi import *

AUTH = UsmUserData('username',
                   authKey='passphrase', privKey='passphrase',
                   authProtocol=usmHMACSHAAuthProtocol,
                   privProtocol=usmAesCfb128Protocol)
TARGET = UdpTransportTarget(('192.168.2.75', 161))

while True:
    for (errInd, errSt, errIdx, varBinds) in getCmd(
        SnmpEngine(), AUTH, TARGET, ContextData(),
        ObjectType(ObjectIdentity('1.3.6.1.4.1.9999.1.1.0')),
        ObjectType(ObjectIdentity('1.3.6.1.4.1.9999.1.4.0')),
        ObjectType(ObjectIdentity('1.3.6.1.4.1.9999.1.6.0')),
        ObjectType(ObjectIdentity('1.3.6.1.4.1.9999.1.7.0')),
        ObjectType(ObjectIdentity('1.3.6.1.4.1.9999.1.8.0')),
    ):
        if errInd or errSt:
            print(f"Error: {errInd or errSt}")
        else:
            v = {str(var[0]): var[1].prettyPrint() for var in varBinds}
            ts = time.strftime('%H:%M:%S')
            print(f"[{ts}]  "
                  f"Температура: {int(v['1.3.6.1.4.1.9999.1.1.0'])/10:.1f}°C  "
                  f"Heap: {v['1.3.6.1.4.1.9999.1.4.0']}%  "
                  f"Протечка: {'ТРЕВОГА' if v['1.3.6.1.4.1.9999.1.6.0'] != '0' else 'ОК'}  "
                  f"Дверь1: {'ОТКРЫТА' if v['1.3.6.1.4.1.9999.1.7.0'] != '0' else 'закрыта'}  "
                  f"Дверь2: {'ОТКРЫТА' if v['1.3.6.1.4.1.9999.1.8.0'] != '0' else 'закрыта'}")
    time.sleep(2)
```

## Сброс к заводским настройкам

Удержание кнопки на GPIO 5 в течение 5 секунд стирает NVS-раздел. После перезагрузки устройство стартует с параметрами по умолчанию из `ethernet_app.c`.
